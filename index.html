FAQ

How many team members can I invite in WDF Projects?

You can invite up to 2 additional members.

Which Technologies are used in WDF Projects?

WDF Projects uses HTML, CSS, and JavaScript for the front-end, PHP for the back-end and MySQL for the database.

Can I use GitHub for WDF Projects?

Yes! You can use GitHub for version control and collaboration on WDF Projects.

How can I get Github Education Pack?

To get the GitHub Education Pack, you need to sign up with a valid educational email address and provide proof of your student status.

Do you provide additional support?

Chat and email support is available 24/7. Phone lines are open during normal business hours.
